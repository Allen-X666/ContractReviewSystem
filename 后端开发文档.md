# 智能合同审查系统 - 后端开发文档

## 一、项目概述

### 1.1 项目简介
智能合同审查助手是一款基于 AI 的企业法务合规解决方案，提供合同上传、智能审查、风险识别、法条关联、报告生成等功能。

### 1.2 技术栈
- **框架**: Spring Boot 4.0.5
- **JDK**: Java 25
- **数据库**: MySQL + Redis
- **AI 框架**: Spring AI 2.0.0-M4
- **模型支持**: OpenAI
- **向量数据库**: PGVector
- **ORM**: MyBatis 4.0.1
- **文档处理**: PDF/DOCX 解析 (Spring AI Document Readers)

### 3.3 合同审查模块（核心）

#### 功能说明
- 异步审查任务处理

- AI 风险分析

#### 审查流程
```
1. 接收审查请求
2. 创建审查任务（状态：pending）
3. 异步执行审查：
   - 阶段1: 文档解析（parsing）
   - 阶段2: 法条检索（retrieving）
   - 阶段3: AI分析（analyzing）
   - 阶段4: 生成报告（generating）
4. SSE 推送进度更新
5. 完成/失败状态更新
```

#### AI 审查服务设计
```java
@Service
public class ContractReviewService {
    
    @Autowired
    private ChatClient chatClient;
    
    @Autowired
    private VectorStore vectorStore;
    
    // 执行审查
    @Async
    public void executeReview(Long reviewId) {
        // 1. 解析文档
        String content = parseDocument(contract);
        updateProgress(reviewId, "parsing", 25);
        
        // 2. 检索相关法条
        List<LawArticle> relatedLaws = retrieveLaws(content);
        updateProgress(reviewId, "retrieving", 50);
        
        // 3. AI 分析风险
        List<RiskItem> risks = analyzeRisks(content, relatedLaws);
        updateProgress(reviewId, "analyzing", 75);
        
        // 4. 生成审查结论
        ReviewResult result = generateResult(risks);
        updateProgress(reviewId, "generating", 100);
    }
    
    // AI 风险分析
    private List<RiskItem> analyzeRisks(String content, List<LawArticle> laws) {
        String prompt = buildAnalysisPrompt(content, laws);
        return chatClient.prompt(prompt)
            .call()
            .entity(new ParameterizedTypeReference<List<RiskItem>>() {});
    }
}
```

```java
@Service
public class LawVectorSearchService {
    
    @Autowired
    private VectorStore vectorStore;
    
    // 初始化法条向量库
    @PostConstruct
    public void initVectorStore() {
        List<LawArticle> articles = lawArticleMapper.selectAll();
        List<Document> documents = articles.stream()
            .map(this::convertToDocument)
            .collect(Collectors.toList());
        vectorStore.add(documents);
    }
    
    // 检索相关法条
    public List<LawArticle> searchRelevantLaws(String query, int topK) {
        SearchRequest request = SearchRequest.query(query).withTopK(topK);
        List<Document> results = vectorStore.similaritySearch(request);
        return convertToLawArticles(results);
    }
}
```

## 五、异常处理

### 5.1 全局异常处理
```java
@RestControllerAdvice
public class GlobalExceptionHandler {
    
    @ExceptionHandler(BusinessException.class)
    public Result<Void> handleBusinessException(BusinessException e) {
        return Result.error(e.getCode(), e.getMessage());
    }
    
    @ExceptionHandler(MethodArgumentNotValidException.class)
    public Result<Void> handleValidationException(MethodArgumentNotValidException e) {
        String message = e.getBindingResult().getFieldErrors().stream()
            .map(FieldError::getDefaultMessage)
            .collect(Collectors.joining(", "));
        return Result.error(400, message);
    }
    
    @ExceptionHandler(Exception.class)
    public Result<Void> handleException(Exception e) {
        log.error("系统异常", e);
        return Result.error(500, "系统繁忙，请稍后重试");
    }
}
```
