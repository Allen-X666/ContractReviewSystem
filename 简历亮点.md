# 智能合同审查系统 - 项目简历

## 项目概述

**项目名称**: 智能合同审查助手 - 企业法务合规 AI 系统  
**项目类型**: 企业级 SaaS 平台  
**技术栈**: Spring Boot + Vue3 + FastAPI + AI/RAG  
**项目规模**: 前后端分离架构，微服务设计

---

## 核心亮点

### 1. 架构设计亮点

#### 1.1 三层异构架构设计
- **前端层**: Vue3 + Vite + Element Plus，现代化响应式界面
- **网关层**: Spring Boot 4.0.5 + Java 25，统一认证鉴权与路由转发
- **AI服务层**: FastAPI + Python，高性能异步 AI 推理服务
- **价值**: 实现了前后端解耦、服务独立扩展、技术栈最优选型

#### 1.2 微服务通信机制
- 设计 SpringBoot 到 FastAPI 的 HTTP 转发机制，统一 API 入口
- 实现服务间超时控制、熔断降级、异常处理策略
- 配置化服务地址管理，支持环境变量动态配置

---

### 2. AI 与智能化亮点

#### 2.1 RAG (检索增强生成) 架构
- **向量数据库**: 基于 ChromaDB 构建法律知识库向量存储
- **嵌入模型**: 集成多类型 Embedding 模型，支持文本语义检索
- **文档检索**: 实现 TTL 缓存机制，支持 LRU 淘汰策略的检索缓存
- **价值**: 合同审查时自动检索相关法条，提升 AI 回答准确性

#### 2.2 LLM 集成与优化
- **模型接入**: 通义千问 (DashScope) 大模型深度集成
- **LangChain 框架**: 构建可复用的审查链 (RiskAnalysisChain/ComplianceCheckChain)
- **Prompt 工程**: 设计结构化 Prompt 模板，支持风险分析、合规检查、合同摘要
- **输出解析**: 自定义 OutputParser，实现结构化数据抽取

#### 2.3 异步审查工作流
- 设计异步 ReviewWorker，支持审查任务后台执行
- 实现任务取消机制，支持用户主动中断审查流程
- 多阶段审查流程：文档解析 → 向量索引 → 风险分析 → 合规检查 → 报告生成

---

### 3. 实时通信亮点

#### 3.1 SSE 实时进度推送
- **技术方案**: Server-Sent Events 实现服务器向客户端单向实时推送
- **架构设计**: SpringBoot 轮询 FastAPI → SSE 推送到前端
- **容错机制**: 连接超时处理、自动重连、异常恢复
- **用户体验**: 审查进度实时可视化，多阶段状态展示

#### 3.2 前端状态管理
- Pinia Store 集中管理审查状态
- 响应式进度条与阶段指示器
- 审查结果实时渲染，支持风险项高亮展示

---

### 4. 后端技术亮点

#### 4.1 Spring Boot 工程实践
- **版本**: Spring Boot 4.0.5 + Java 25 (最新 LTS)
- **数据访问**: MyBatis + PageHelper 分页插件
- **API 文档**: SpringDoc OpenAPI (Swagger) 自动生成接口文档
- **缓存**: Redis 集成，支持分布式缓存与会话管理

#### 4.2 安全与认证
- **JWT 认证**: 基于 Token 的无状态认证机制
- **Token 刷新**: 实现 Token 自动续期，无感知登录保持
- **接口鉴权**: 拦截器实现统一权限校验
- **限流防护**: RateLimitService 防止接口滥用

#### 4.3 文件存储与处理
- **OSS 集成**: 阿里云 OSS 对象存储，支持大文件分片上传
- **异步处理**: @Async 线程池实现邮件发送等非阻塞操作
- **文件预览**: PDF/DOCX 文件在线预览与下载

#### 4.4 代码规范
- 统一响应格式封装 (Result/ResultCode)
- 全局异常处理 (GlobalExceptionHandler)
- 枚举类型处理器 (MyBatisEnumTypeHandler)
- 常量集中管理

### 5. 前端技术亮点

#### 5.1 Vue3 生态深度应用
- **框架**: Vue3 Composition API + `<script setup>` 语法
- **构建工具**: Vite 5.0，极速开发体验
- **UI 组件**: Element Plus 2.5，企业级组件库
- **状态管理**: Pinia 2.1，类型安全的 Store

#### 5.2 文件预览系统

- **DOCX 预览**: mammoth.js 实现 Word 文档转 HTML 预览
- **通用预览**: 统一 FilePreviewDialog 组件，支持多格式

#### 5.3 安全与体验
- **安全存储**: crypto.js 实现敏感数据加密存储
- **Token 管理**: 自动刷新机制，无感登录续期
- **响应式设计**: 适配多终端的界面布局
